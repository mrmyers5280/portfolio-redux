Colorado Surface Water Conditions

I use the Colorado Surface Water Conditions website a lot in the summer to find the best place to fish and I thought it could use an update. This project uses the Colorado Surface Water Conditions API and D3.js to display a line graph of water levels for a selected water gauge. Some of the challenges I faced were learning how to use the API, storing the data in a MongoDB database, and working with D3.js. My responsibilities on this project were as the developer and designer.

Quiz App

A short, movie-themed quiz that allows the user to select an answer and click a submit button to move to the next question. The interface displays the question number of the number of questions the user is on. At the end of the quiz, the user's number of correct questions is displayed. The quiz can be restarted at any time using the reset quiz button. One of the challenges I faced was using the 'this' keyword and understanding what it referred to. My responsibilities on this project were as the developer and designer.

Sights & Sounds

An API mashup using the Instagram API along with the Soundcloud API. The user enters an Instagram tag to search for and clicks the Start button. The page then loads the images with the specified tags. The app also displays the Soundcloud player widget that randomly selects a song from the 'chill' genre. The user must push play because auto-playing music can be annoying. Writing the JavaScript to get the next batch of images was particularly challenging. My responsibilities on this project were as the developer and designer.

Shopping List

This project allows the user to build a shopping list (or to-do list). It uses jQuery to add, check and uncheck, reorder and remove items from the list. Although it is a relatively simple project, I very much enjoyed it and so thought it should be featured. I really enjoyed coming up with the design of a note on a refrigerator. And I enjoyed even more working with jQuery and how easy it was to work with. My responsibilities on this project were as the developer and designer.
